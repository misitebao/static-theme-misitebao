## 项目介绍

本项目是一个适用于博客网站的静态模板，可用于任意模板引擎。

## 功能

- 多颜色主题切换
- 多种代码高亮风格
- 支持用于任意模板引擎
- 支持任意静态网站生成器

## 快速入门

安装依赖：

`npm install`

本地开发预览模式：

`npm run dev`

构建前端资源：

`npm run build`

以本地服务方式预览前端资源：

`npm run serve`
